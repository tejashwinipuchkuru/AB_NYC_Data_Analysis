# Airbnb Data Cleaning Project

## Week 1 Internship Task - Data Cleaning

### Project Overview
This project focuses on cleaning and preprocessing the Airbnb dataset to make it suitable for further analysis and machine learning applications.

### Dataset
Dataset used: Airbnb NYC Dataset

### Data Cleaning Steps Performed

1. Loaded dataset using Pandas
2. Checked dataset structure and information
3. Handled missing values
4. Removed duplicate records
5. Fixed incorrect data types
6. Converted price column into numerical format
7. Converted date columns into datetime format
8. Saved cleaned dataset

### Technologies Used

- Python
- Pandas
- Jupyter Notebook

### Files Included

- data_cleaning.ipynb
- AB_NYC_Cleaned.csv
- README.md

### Outcome

The final cleaned dataset is ready for Exploratory Data Analysis and visualization.
